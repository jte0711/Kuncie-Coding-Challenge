# Kuncie-Coding-Challenge

### Supported Device:

- Pixel 5, API 32

### Supported Features:

- Search Artist
- Play/Pause Songs
- Media Player

### Requirement

- Node v16.13.1

### How to Run

- Install all the dependecies with `yarn`
- Open Android Simulator device
- Run `yarn android`
